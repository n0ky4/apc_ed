# APC - ED

Este repositório é meu histórico de estudos e exercícios relacionados à disciplina de [Algoritmos e Programação de Computadores (APC)](apc/) e [Estruturas de Dados (ED)](ed/).

O diretório [bin/](bin/) contém scripts e utilitários para facilitar a execução e comparação de programas, como:

- `runc <arquivo.c>`: Compila e executa programas em C
- `runcpp <arquivo.cpp>`: Compila e executa programas em C++
- `runcompare <arquivo1.c> <in.txt> <expected.txt>`: Compila e executa um programa em C, comparando a saída com um arquivo esperado.
